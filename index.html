Digestor

This is a tiny app that parses an existing podcast feed of a radio show on Radio Študent and improves it to make it fully compatible with the requirements of iTunes, Spotify and other podcast platforms.

Relevant links:

  • Feed - the main thing generated by this app
  • Pritiskavec gold - radio show in question
  • Radio Študent - station where the show originally airs
  • mzgajner/digestor - repo with the source code of this app
